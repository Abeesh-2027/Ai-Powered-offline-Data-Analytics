🚀 AI powered offline data Analytics 

An advanced offline AI-powered data analytics platform built with Streamlit.
Upload your dataset, clean it automatically, visualize insights, and ask questions using a local AI model — all without internet access.

🔥 Features
📂 Upload CSV datasets بسهولة
🧹 Automatic data cleaning (missing values, duplicates, formatting)
📅 Smart date detection & standardization (YYYY-MM-DD)
🔍 Advanced filtering (numeric & text-based)
📊 KPI dashboard (Total, Avg, Max, Min)
📈 Multiple visualizations (Line, Bar, Pie, Scatter, Histogram, Box, Area)
🤖 AI-powered insights using local LLM (Ollama)
💬 Ask questions in natural language
🧠 Tech Stack
Frontend: Streamlit
Backend: Python
Libraries: Pandas, NumPy, Matplotlib
AI Engine: Ollama (LLaMA3)
Report Generation: ReportLab
⚙️ Installation
pip install streamlit pandas numpy matplotlib reportlab ollama
▶️ Run the App
streamlit run app.py
